# Bearish Den Image Stitcher

Chrome/Brave extension to stitch your Bearish Den images into a single grid image.

## Features
- Stitch My Bears or Hibernating
- Option to include both tabs
- Tight grid or perfect square layout
- One-click PNG download

## Installation (Chrome / Brave)

1. Download this repo as a ZIP or clone it
2. Unzip the folder
3. Open:
   chrome://extensions  
   or  
   brave://extensions
4. Enable **Developer mode**
5. Click **Load unpacked**
6. Select the unzipped folder

## Usage

1. Go to https://www.bearish.af/den
2. Click the Bearish Stitcher icon
3. Choose layout options
4. Click Stitch Images
